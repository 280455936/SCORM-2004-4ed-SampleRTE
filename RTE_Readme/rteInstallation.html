Installation


Prerequisites

IMPORTANT: Read ALL of the Installation Instructions BEFORE beginning installation.

This release of the SRTE requires the Java Development Kit (JDK) from any one of the following Java versions:

  • Java SE 6, Update 1 or later

The latest version of this software can be downloaded free of charge at http://java.sun.com/javase/downloads/index.jsp. Archived versions are available at http://java.sun.com/products/archive/. Refer to this site for a list of specific Java versions verified.

Notes:

  • The SRTE requires the full Java Development Kit. The Java Runtime Environment (JRE) alone is NOT sufficient for this software.
  • The SRTE client may be accessed from a machine other than the one on which it is installed using Microsoft® Internet Explorer®. In this case, the Java Runtime Environment must be installed on the second machine.
After installing the Java Development Kit, you must:
  1. Create an environment variable named "JAVA_HOME" and set its value to the root directory of the Java Development Kit.
    • For Windows XP: Right click "My Computer", select "Properties", select the "Advanced" tab and then click "Environment Variables". Add a JAVA_HOME entry in the User or System Variables that points to the location where you installed the Java Development Kit.
    • For Windows Vista: Open the "Start" menu and right click "Computer", select "Properties", select the "Advanced System Settings" link on the left navigation menu, and then click Environment Variables. Add a JAVA_HOME entry in the User or System Variables that points to the location where you installed the Java Development Kit.

  2. Add the full path of the Java Development Kit bin directory (where the java.exe is located) to the PATH environment variable.
    • For Windows XP: Right click "My Computer", select "Properties", select the "Advanced" tab and then click "Environment Variables". Add the Java Development Kit bin directory to the Path by adding a semicolon followed by a reference to the JAVA_HOME environment variable followed by "\bin" (i.e., %JAVA_HOME%\bin).
    • For Windows Vista: Open the "Start" menu and right click "Computer", select "Properties", select the "Advanced System Settings" link on the left navigation menu, and then click Environment Variables. Add the Java Development Kit bin directory to the Path by adding a semicolon followed by a reference to the JAVA_HOME environment variable followed by "\bin" (i.e., %JAVA_HOME%\bin).
Constraints

As of this release, the SRTE was tested on the Intel® Pentium® PC platform running Microsoft® Windows® Vista SP1 and XP Professional SP3. Microsoft Internet Explorer 7.0 and Internet Explorer 8.0 were also tested with the SRTE. The SRTE was not formally tested with any other browsers on any other platforms than those indicated above.

Installation

IMPORTANT: Only install the SRTE after installing the Java Development Kit as described in the Prerequisites.

The SRTE is available for download as a self-extracting executable install file. Once you have completed downloading the file, perform the following steps:

  1. Double-click the downloaded self-extracting executable file to begin the installation process.
  2. Click Next to continue from the "Welcome" page.
  3. Read the "License Agreement"
    • Click Yes to agree and continue with the install.
    • Click No if you do not agree with the terms of the license and wish to exit the install.
  4. Choose whether you would like the SRTE to be installed for Anyone who uses this computer (all users) or Only me.
  5. The recommended root installation directory is "C:\ADL". You may choose another root installation directory, but the SRTE install will always append the "\ADL\SCORM_2004_4th_Ed_Sample_RTE_V1.1.1" sub-directory to the specified root installation directory. The remainder of this Readme document assumes that the SRTE will be installed to "C:\ADL\SCORM_2004_4th_Ed_Sample_RTE_V1.1.1". This is referred to as "<drive>:\<install_directory>". If you override the default to use a different root installation directory, please remember the directory you selected when reading the remainder of this Readme.
  6. Click Next
  7. Click Install to begin the installation
  8. You will have the option to view a copy of this Readme document.
    • Check "Show the Readme file" to view the Readme.
    • Uncheck "Show the Readme file" to skip viewing the Readme.
  9. Click Finish to complete the installation.
Note: This installation will not remove or un-install previous versions of the SRTE.

Note: The apache-tomcat-5.5.20 servlet engine and web server will be installed and pre-configured to bind to port 8080.

Configuration
After installation, perform the following actions to ensure that the SRTE runs properly:
  • The Internet Explorer browser must be set to check for a new page every visit to the page. To configure the browser:
    • Using Microsoft Internet Explorer 6.0 Service Pack 2
      1. Open a Microsoft Internet Explorer browser
      2. Go to "Tools" and select "Internet Options"
      3. Select the "General" tab, then under "Temporary Internet files", click Settings
      4. Select the "Every visit to the page" radio button if it is not already selected
      5. Click OK and then OK again.

    • Using Microsoft Internet Explorer 7 and 8
      1. Open a Microsoft Internet Explorer browser
      2. Go to "Tools" and select "Internet Options"
      3. Select the "General" tab, then under "Browsing History", click Settings
      4. Select the "Every time I visit the page" radio button if it is not already selected
      5. Click OK and then OK again.

  • The Java Plug-in must be configured to disable caching. To disable the Java Plug-in cache:
    • For Java SE 6
      1. Open the Windows "Control Panel"
      2. Double click the "Java" icon
      3. Select the "General" tab
      4. Under the "Temporary Internet Files" section, click Settings
      5. Uncheck the "Keep Temporary Files on my computer." if it is checked.
  • Set the Sun Java Plug-in as the default Java Environment for the Microsoft Internet Explorer browser by doing the following:
    • For Java SE 6
      1. Open the Windows "Control Panel"
      2. Double click the "Java" icon
      3. Select the "Advanced" tab
      4. Expand the "Default Java for Browsers" section
      5. Select the checkbox next to the browser you are using to run the SRTE
      6. Click Apply.
  • Perform these additional actions if you are using Windows XP SP2 or Vista:
    • Allow JavaScript to run by:
      1. Opening a Microsoft Internet Explorer browser
      2. Clicking "Tools" and then "Internet Options"
      3. On the "Advanced" tab under the "Security" section, check the box that states "Allow active content to run in files on My Computer"
    • Allow the SRTE to pop-up windows by either disabling pop-up blocking or exclude the SRTE from pop-up blocking.

This completes the installation of the SRTE.

Uninstalling the SRTE

Use one of two possible methods to uninstall the SRTE:

  • For Windows XP, use the Windows Control Panel "Add/Remove Programs" feature:
    1. Open the Windows "Control Panel"
    2. Double-click the "Add or Remove Programs" icon
    3. Highlight "SCORM 2004 4th Ed. Sample Run-Time Environment Version 1.1.1"
    4. Click Remove and follow the instructions
  • For Windows Vista, use the Windows Control Panel "Programs and Features" option:
    1. Open the Windows "Control Panel"
    2. Double-click "Programs and Features"
    3. Highlight "SCORM 2004 4th Ed. Sample Run-Time Environment Version 1.1.1"
    4. Click Uninstall on the toolbar above the list and follow the instructions
  • If this version of the SRTE is already installed, running the install again will uninstall the software. This method will not re-install the SRTE. You will need to run the install a third time to accomplish a re-installation.
Note: The Sample Run-Time Environment may not appear in the "Add/Remove Programs" menu. If it does not appear, use the second method to uninstall it.

These methods of uninstalling will not remove files or directories that have been altered in the course of running the SRTE. To complete the process of uninstalling the SRTE, the follow these steps:
  1. Delete the directory that the SRTE was installed to (<drive>:\<Install Directory>\SCORM_2004_4th_Ed_Sample_RTE_V1.1.1 by default) along with any files or directories contained within it
  2. Delete the <drive>:\SCORM4EDSampleRTE111Files directory
