Environment Variable
  • The Environment Variable will be set when the SRTE is installed. If it must be manually reset, please do the following:
    1. Right-click "My Computer" and select "Properties"
    2. Click the "Advanced" tab
    3. Click Environment Variables
    4. Click New
    5. Set the variable name to SCORM4ED_SRTE111_HOME
    6. Set the variable value to <drive>:\<install_directory>
    7. Click OK on the New Variable window
    8. Click OK on the Environment Variables window
    9. Click OK on the Systems Properties window
