Sample RTE Administrative Options Help
The Sample RTE Administrative Options section enables an administrator to perform the administrative actions available the SRTE.

Administrative Options
  • Import Course - Allows an administrator to import a course into the SRTE. more
  • Import Multiple Courses - Allows an administrator to do a batch import of courses into the SRTE. more
  • Manage Courses - Allows an administrator to edit comments available for a course. more
  • Add Users - Allows an administrator to add users to the SRTE. more
  • Delete Users - Allows an administrator to delete a user from the SRTE. more
  • Manage Users - Allows an administrator to edit user information in the SRTE. more
  • All User's Course Status - Allows an administrator to view user status for a course. more
  • Global Objectives Administration - Allows an administrator to view/edit global objectives in the SRTE. more
  • Clear Database - Allows an administrator to clear all information stored in the SRTE databases including user status, courses, and objectives information.
"Show/Hide" Functionality
  • Clicking the Show/Hide links on any given section will show or hide everything in the section except the title of the section.
