Sample RTE Available Courses Help
The Sample RTE Available Courses section enables a user to register for courses that are currently available in the SRTE.

Registration Instructions
  1. Select the checkbox next to the courses you wish to modify in the Available Courses section. All courses can be selected or unselected by checking the 'All' checkbox.
  2. Click one of the following buttons to modify the course:
    • Register - Registers the selected courses can makes them available for the user to launch.
    • Delete Course Delete allows an administrator to disable and remove access to a course that was imported into the SRTE.
      *Note: Disabled courses are not deleted from the SRTE. All courses may be deleted by choosing the Clear Database option.
"Show/Hide" Functionality
  • Clicking the Show/Hide link on the section will show or hide everything in the section except the title of the section.
