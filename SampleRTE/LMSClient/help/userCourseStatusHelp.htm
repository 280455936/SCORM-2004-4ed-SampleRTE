View All User's Course Status Help

View All User's Course Status allows the administrator to choose a user and view the status of any course in which that user is registered.

View All User's Course Status Instructions
  1. Choose the user whose course status you would like to view and click Submit.
  2. Select the radio button next to the course whose status you would like to view.
  3. Click Submit to view the status of the selected course.
