Sample RTE Registered Courses Help
The Sample RTE Registered Courses section enables a user to launch, view status of the course, or unregister for courses in the SRTE.

Registration Instructions
  1. Select the checkbox next to the courses you wish to modify in the Registered Courses or Available Courses section. All courses can be selected or unselected in a given section by checking 'All' checkbox in each section.
  2. Click one of the following buttons to modify the course:
    • Reset - Resets the selected courses and resets all data (excluding Global Objectives data) associated with the courses.
    • UnRegister - UnRegisters the selected courses and removes all data (excluding Global Objectives data) associated with the courses.
    • Delete Course Delete allows an administrator to disable and remove access to a course that was imported into the SRTE.
      *Note: Disabled courses are not deleted from the SRTE. All courses may be deleted by choosing the Clear Database option.
Launch Course Instructions

If you would like to begin a new attempt on a course, please select Start Course.

Note: This option will not be available if you are required to choose an activity to begin the course; in these cases please select View Table of Contents.

If you suspended your previous attempt on a course, please select Resume Course to continue that attempt.

If you would like to choose an activity to begin a course, please select View Table of Contents to see which activities are available. After viewing the Table of Contents, please choose an activity to begin the course.

Note: This option will not be available if the course does not provide a Table of Contents; in these cases please select Start Course or Resume Course.
Show/Hide Course Status Instructions

Show/Hide Course Status allows the users to view their status in a course including if the course was completed or not completed, passed or failed, and the score for the course.

  • Click the "Show/Hide Course Status" link below the course whose status you would like to view.
"Show/Hide" Functionality
  • Clicking the Show/Hide links on any given section will show or hide everything in the section except the title of the section.
