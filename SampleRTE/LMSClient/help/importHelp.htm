Course Import Help

Course Import allows an administrator to import a SCORM® Conformant Content Aggregation Package into the SRTE.

Import Instructions
  1. Click Browse and select the Zip file containing your course. This Zip file MUST BE a valid SCORM 2004 4th Edition Content Aggregation Package.
  2. Select the radio button indicating whether you would like to Fully or Partially validate the course. Click Submit.

    NOTE:
    • Full - An Internet connection is required to fully validate the course
    • Partial - An Internet connection is not required. However, if the content package references online controlling documents or online file resources, errors may be displayed during the import process but will be ignored. Other errors may be displayed which prevent the course from importing. If you choose to not fully validate the course and it contains errors and still imports, it may behave unexpectedly.
