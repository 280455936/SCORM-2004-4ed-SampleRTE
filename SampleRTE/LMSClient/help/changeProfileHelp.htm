User Preferences Help
User Preferences Administration allows users to edit or change their password and set additional learner preferences, which include audio_level, language, delivery_speed and audio_captioning.

Edit User Profile Instructions
  1. To change the current password, select the text field and enter the new password in the password text field and click Submit.
  2. To modify the learner_preference.audio_level, select the audio_level preference text box and enter a real number 0..*.
  3. To update the cmi.learner_preference.language, select the language text box and enter a valid language code.
  4. To update the cmi.learner_preference.delivery_speed, select the delivery speed text box and enter a valid real number 0..*.
  5. To update the cmi.learner_preference.audio_captioning, select the audio_captioning text box and enter a valid choice.
    • -1 represents the "off" state
    • 0 represents "no change" state
    • 1 represents the "on" state
  6. Click Submit.
