Manage Users Help

Manage Users allows an administrator to view and edit the information associated with a given user.

Manage Users Instructions
  1. Select the name of the user whose information you would like to view or edit and click Submit.
  2. The current password displayed in the password text field can be modified and or changed by selecting the text field and modifying as needed. After selecting Submit, the new password will be changed to the updated text within the password text box or kept the same if no modifications were made.
  3. To modify the learner_preference.audio_level select the audio_level preference text box and enter a real number 0..*.
  4. To update the cmi.learner_preference.language select the language text box and enter a valid language code.
  5. To update the cmi.learner_preference.delivery_speed select the delivery speed text box and enter a valid real number 0..*.
  6. To update the cmi.learner_preference.audio_captioning select the audio_captioning text box and enter a valid choice.
    • -1 represents the "off" state
    • 0 represents "no change" state
    • 1 represents the "on" state
  7. To modify the users access rights, select either Admin or User from the dropdown control.
  8. Click Submit.
