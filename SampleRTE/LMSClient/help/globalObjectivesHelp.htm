Global Objectives Administration Help
Global Objectives allow progress information for a learner to be shared by more than one SCO. For more information on Global Objectives, refer to SCORM 2004 4th Edition.

Global Objective Administration Instructions
  1. New Global Objectives may be created or existing Global Objectives may be modified.
  2. To create a new global objective, select Create Global Objectives, fill in the necessary information, and click Submit.
  3. To view or modify an existing global objective, click "View or modify an existing global objective." Choose the user whose global objectives you would like to view or modify, and click Submit.
  4. Select the course whose Global Objectives you would like to view. All objectives not associated with a course may also be viewed. If you choose a course and no global objectives are associated with that course, all global objectives that are not associated with any course will be displayed.
  5. Click Submit.
  6. To modify global objectives, select the objectives to delete or reset individually or use the Select All option to select all of the objectives. Choosing Clear Selection allows you to deselect objectives that were already selected.
  7. Click Submit.
