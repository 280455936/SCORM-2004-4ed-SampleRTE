[INVALIDEVENT] page



NOTES:
This [INVALIDEVENT] page indicates that the ADL Sequencer received a navigation event that is not permissible. Look at the User Interface controls present during course execution and evaluate whether they should be visible.

Typical reasons this page is being encountered include:
  • A Continue Navigation Request was invoked within a flow = False cluster.
  • A Previous Navigation Request was invoked from the root of the Activity Tree.
  • A Previous Navigation Request was invoked within a forward-only cluster.
  • A Choice Navigation Request targeted an activity within a choice = False cluster.
  • A Choice Navigation Request targeted an activity that is hidden from choice.
