[NO ACTIVITY TO DELIVER] page



NOTES:
This [NO ACTIVITY TO DELIVER] page indicates that the ADL Sequencer traversed the conceptual activity tree to a point where there is no activity to deliver. This is not necessarily an error. For instance, the sequencing process may not deliver any content, but may indicate to the LMS RTE service that a menu must be displayed.

Typical reasons this page is being encountered include:
  • The Current Activity terminated without indicating the next Navigation Request to be processed.
  • A Navigation Request was invoked that does not result in identifying an activity for delivery and no other exception was identified.
  • The attempt to terminate the Current Activity resulted in an exception. This error will only occur if the sequencer entered an unstable state caused by the SRTE closing unexpectedly.
